плохо, что в видео показывают через IDE. Надо сначала с гитом разобраться.  Может кому понадобиться:
1) Есть проект(папка), которая клонирована на комп с github
2) Создаем новую ветку git branch name new branch(например git branch HW04) - обязательно проверяем в командной строке, что мы находимся в нужной папке!!!!!!!
3) Переходим в новую ветку git chekout HW04
4) Создаем каталог для нашей домашки mkdir HW04
5) Переходим в этот каталог cd HW04
6) Создаем файлы с домашкой touch task12.py и тд.
7) пишем код
8) Затем комитем  git commit -m "task done"
9) Загружаем в репозиторий git fetch
10) Заходим в свой github,  переходим в пулреквест, делаем новый пулреквест там выбираем   в правом указателе веток, нашу новую ветку - делаем новый пулреквест.
11) В адресной строке браузера копируем адресс пулреквеста
12) Идем на портал GB, где надо сдавать ДЗ. В поле для комментариев вставляем скопированный адрес. 
13) Нажимаем отправить. 
14) Идем пить чай, ждем обратной связи от преподавателя :)
